##安裝虛擬環境
1. 安装 Python
2. 安装 virtualenv : pip install virtualenv
3. 创建虚拟环境 : virtualenv myenv
4. 激活虚拟环境 : myenv\Scripts\activate
5. 在虚拟环境中安装包 : pip install package_name
6. 退出虚拟环境 : deactivate

'''
