-git clone 網址 是下載檔案在自己電腦裡面(這只會一到兩次)

-git pull 適用在當在其他裝置上更改檔案時，可以抓下來同步檔案(這會用比較多次)
''' -安裝虛擬環境

安裝 Python
安裝 virtualenv : pip install virtualenv
建立虛擬環境: virtualenv myenv
啟動虛擬環境: myenv\Scripts\activate
在虛擬環境中安裝套件: pip install package_name
退出虛擬環境: deactivate

''' -在虛擬環境裡安裝套件
